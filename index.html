Website

The website has gotten another major update! A new tab has been added for Animations & Comics and the other tabs have been rearranged.

UltraEd64

I'm working towards a full tutorial series for UltraEd that will cover making a platformer and a shooter. The first part of which I'm hoping to upload soon, but we'll see how that goes. Please keep in mind that the software is constantly being updated, so some parts of the tutorials may become outdated. But I will try to include a precompiled copy of the version of the engine I use with each video.

There are still quite a few things that have to be fixed before a closed test and review can be done, mainly the battle system. The project is currently in it's final proposed generation, G10.

O.D.S

ODS has been moved off of hold and a re-work of the game mechanics is currently in development.

Fetch Quest
