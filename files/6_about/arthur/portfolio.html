About Me

I have a passion for game design, hardware development, and creating fun experiences and stories for people to enjoy. Since I was 9 years old I have always wanted to make video games for others, so I dedicated my time to teaching myself how to develop them. From programming to stories, I have learned a ton about every aspect of building an experience from the ground up, and I'm always willing to learn more to create the best experiences possible!

NUPDK Software

The Neverway Unity Project Development Kit, or NUPDK for short, is a Unity package that contains everything you may need for getting started with creating a Neverway project in Unity 2019 or later. NUPDK acts as an excellent starting point for any project from mobile to virtual reality. Included are templates for a firstperson shooter, a topdown RPG, our own custom input system (that even has VR and Mobile support!), a sound/music manager, and a game state saver.

Tags: Unity, Tool

Engine: Unity 2021

Platforms: Windows, Linux, MacOSX, Android

Programming (C#)
