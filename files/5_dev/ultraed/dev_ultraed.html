Controller inputs

To get data from the controller we use the function nuContData[#] . The # represents which players controller the script will read from.

All controller inputs should be read from the input loop void $input(NUContData gamepads[4]) { } , which should be included in each new actor script.

So if we wanted to get inputs from player 1, we would type something like this:

void $input(NUContData gamepads[4])

{

     nuContData[0]

}


To get the button inputs from the controller we use different variables of nuContData[#].button In an if statement. The variables for each button input are as follows:

A: A_BUTTON

B: B_BUTTON

Start: START_BUTTON

Z: Z_BUTTON

L: L_BUTTON

R: R_BUTTON

(Ps I don't currently know what the input code is for the d-pad but I will add it when I figure it out. -Liz)

So if we wanted to get an A button press from player 1, we would type something like this:

void $input(NUContData gamepads[4])

{

     if (nuContData[0].button && A_BUTTON)

     {

          // Run some code

     }

}


Getting inputs from the analog stick is a bit different. Instead of using nuContData[#].button we will be using nuContData[#]-> and to define the stick we use these:

X Axis: stick_x

Y Axis: stick_y

The x and y axis should be defined as two seperate float variables. So if we wanted to get left on the analog stick from player 1, we would type something like this:

float stickX;

void $input(NUContData gamepads[4])

{

     if (stickX >= 1)

     {

          // Run some code

     }

     stickX = nuContData[0].stick_x;

}

You can also apparently use stickX = nuContData->stick_x; , but this will always only default to getting the value of the first player.
